# これはなに?
Micronaut(まいくろのーと)でサーバーサイドKotlinに入門するサンプルコードです。

サンプルその2として、入門から一歩進んでDBへのアクセスや単体テストが動くようになっています。

詳細は技術書典 17の新刊「**Micronautで入門!実践!サーバーサイドKotlin**」で!
